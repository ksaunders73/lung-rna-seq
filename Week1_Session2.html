Standard Preprocessing

Filtering Cells

The number of genes and UMIs (nGene and nUMI) are automatically calculated for every object by Seurat. For non-UMI data, nUMI represents the sum of the non-normalized values within a cell We calculate the percentage of mitochondrial genes here and store it in percent.mito using AddMetaData. We use object@raw.data since this represents non-transformed and non-log-normalized counts The % of UMI mapping to MT-genes is a common scRNA-seq QC metric.

mito.genes <- grep(pattern = "^MT\\.", x = rownames(sobj@assays[["RNA"]]), value = TRUE)
print(mito.genes)
##  [1] "MT.ND1"  "MT.ND2"  "MT.CO1"  "MT.CO2"  "MT.ATP8" "MT.ATP6" "MT.CO3" 
##  [8] "MT.ND3"  "MT.ND4L" "MT.ND4"  "MT.ND5"  "MT.ND6"  "MT.CYTB"
percent.mito <- Matrix::colSums(sobj@assays[["RNA"]][mito.genes, ])/Matrix::colSums(sobj@assays[["RNA"]])

Adding Metadata

AddMetaData adds columns to object@meta.data, and is a great place to stash QC stats Seurat v2 function, but shows compatibility in Seurat v3.

sobj <- AddMetaData(object = sobj, metadata = percent.mito, col.name = "percent.mito")

In case the above function does not work simply do:

sobj$percent.mito <- percent.mito

Quality Control Visualizations

GenePlot is typically used to visualize gene-gene relationships, but can be used for anything calculated by the object, i.e.Â columns in object@meta.data, PC scores etc. Since there is a rare subset of cells with an outlier level of high mitochondrial percentage and also low UMI content, we filter these as well.

Filtering Cells

We filter out cells that have unique gene counts (nFeature_RNA) over 2,500 or less than 200 Note that > and < are used to define aâgateâ.
-Inf and Inf should be used if you donât want a lower or upper threshold.

sobj <- subset(x = sobj, subset = nFeature_RNA > 200 & nFeature_RNA < 2500 & percent.mito >  -Inf & percent.mito < 0.05 )

Log Normalize

After removing unwanted cells from the dataset, the next step is to normalize the data. By default, we employ a global-scaling normalization method âLogNormalizeâ that normalizes the gene expression measurements for each cell by the total expression, multiplies this by a scale factor (10,000 by default), and log-transforms the result.

sobj <- NormalizeData(object = sobj, normalization.method = "LogNormalize", scale.factor = 10000)
